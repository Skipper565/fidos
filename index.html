    Školní projekt ohledně značkovacích jazyků, webová implementace "Dialogového vyhledávání dat v souboru ve formátu Open Document Spreadsheet"

    Project maintained by Skipper565 Hosted on GitHub Pages — Theme by mattgraham

    Welcome to the FIODS web pages!

    The acronym stands for Finding In ODS - the main purpose of this PB138 school project is to develop a web application, which will allow the user to search a file system stored in an ODS file (Open Document Spreadsheet). The application should have a form of a dialog system capable of understanding natural language.

    A programming language of choice is Java with interfaces built using W3C Voice Browser Activity.

    Here is the original assignment:

    "Studenti navrhnou a implementují webový dialogový systém v jazyce Java s rozhraním popsaným pomocí standardů W3C Voice Browser Activity, který umožní pomocí dotazů v přírozeném jazyce (hlasově nebo textově) prohledávat souborovou kartotéku v souboru ve formátu Open Document Spreadsheet. Soubor může obsahovat více tabulek, které jsou pojmenovány, a které obsahují strukturovaná data, vždy jeden záznam na jednom řádku v tabulce. 1. řádek dané tabulky obsahuje popis jednotlivých atributů, podle kterých lze vyhledávat. Ukázková data dodá zadavatel."
